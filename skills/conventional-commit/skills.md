# Conventional Commit
## Cuando generes un mensaje de commit:
feat(reserva): cancelar una reserva 
feat(reserva): confirmar una reserva 
fix(vehiculos): rechazar vehiculos sin patente
fix(vehiculos): rechazar vehiculos sin tipo
docs(reserva): indicar motivo del rechazo
docs(vehiculos): indicar motivo del baja definitiva
